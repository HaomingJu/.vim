# Terminal工具汇总

## Tig:
* 安装方法:
```
sudo apt-get intall tig
```
* 说明: git log 更加友好的浏览器.

## fzf:
* 安装方法:
```
git clone https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install
```
* 说明: 命令行下的模糊查找.





# Python工具汇总
## ici
* 安装方法:
```
sudo pip install ici
```
* 说明: Python英文翻译工具.

## bpython
* 安装方法:
'''
sudo pip install bpython
'''

* 说明: 更加友好的Python交互式解释器.
